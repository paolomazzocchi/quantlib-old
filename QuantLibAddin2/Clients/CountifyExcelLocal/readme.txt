Save.xls: To be loaded into a running instance of the Rate Curve Framework.  Saves the data to GBP_market.xml.

GBP_Market.bat:  Starts a new empty Excel session and loads the XLL, GBP_Market.xlsm, and GBP_market.xml.  Simulates a workbook which acts as a front end to a server process caching the data.  A local process which simulates the behavior of the =countify process.
